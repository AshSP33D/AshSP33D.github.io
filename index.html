ASH'S PORTFOLIO

This project was tasked to me by my drum teacher. While initially developing it in Python, and writing a HTML file, I decided to switch over to constructing the page from within the browser using JavaScript for user convenience. It makes use of a custom font, results in 10, 16-beat bars and can be printed.
Being developed off the other drum sequences project, I simply removed some pieces of code I felt weren't needed any more (as I was creating my own sequences to go into the array) and replaced the sequences to replace with the Basic Rock variety. I decided to reduce the amount of checking the code had to do and the system I used for the arrays. In depth explanations can be found within the HTML files themselves.
I undertook this project myself with the sole intention of being able to host some of my small experiments (regardless of whether to get them to work in this context it is close to impossible) on a Wordpress site. I quickly learned that you can only run JS, which is required, on premium Wordpress servers, which meant that I had to build the page on the only premium server I have access to. I felt it was a nice challenge, and could be used for other purposes by other people (for example for hosting a portfolio without having to purchase multiple different addresses). Of course, GitHub is still a better place to do the same function.
